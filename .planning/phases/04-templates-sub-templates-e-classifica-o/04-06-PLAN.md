## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-12 | Tampering (XSS) | render de nomes de template/valores extraídos | mitigate | React renderiza como texto puro; NENHUM `dangerouslySetInnerHTML` (T-02-11 estabelecido); grep de verificação |
| T-04-18 | Information Disclosure | erro de API exibido na UI vaza detalhe interno | accept | Mensagens de erro do UI-SPEC são genéricas pt-BR ("Não foi possível…"); single-tenant local; sem dado sensível em mensagem de erro |
| T-04-19 | DoS | regex do operador digitada no form | mitigate | O form só ENVIA a regex como string; a aplicação segura (fullmatch + limite) é backend (Plan 02); o frontend não executa a regex do operador |
